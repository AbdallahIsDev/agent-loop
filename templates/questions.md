# Questions

Two-way communication between you and the user.

## Format

### Q001 — Short question summary

**From:** Agent | User
**Round:** (round number)
**Priority:** High | Medium | Low
**Status:** Open | Answered | Expired
**Expiry:** (round number when question expires if unanswered)
**Context:**
What prompted this question.

**Question:**
The actual question.

**Answer:**
Filled when responded to.
